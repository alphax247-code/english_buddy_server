TROUBLESHOOTING
---------------
- View error logs: Web tab > "Error log" link
- View access logs: Web tab > "Access log" link
- If you get "ModuleNotFoundError": make sure your venv is set
  correctly and all requirements are installed
- If you get 502/504 errors: check the error log; likely an import
  error or a missing .env value
- Free accounts cannot reach external APIs (Paysuite):
  upgrade to a paid plan
- The background polling task (_poll_pending_payments) uses asyncio
  which runs fine under uvicorn but PythonAnywhere uses a WSGI worker.
  The task will still start via the lifespan handler since
  PythonAnywhere runs uvicorn under the hood for ASGI apps.
